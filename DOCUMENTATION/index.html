Please feel free to contact us if you have questions or problems about this template. No guarantee is provided, but we will do our best to provide you with the best possible assistance.

Quickstart guide.

You can work with the template directly after unzipping it. We recommend rough order of actions which would help you to customize it according to your needs.

  1. Unzip archive.
  2. Choose page which you will use in project. Rest page you can remove from working directory.
  3. Change main menu code (accordingly to pages)
    1. Delete menu item and nesting links "Feature". It contain page for crafting needs.
    2. Choose suitable variant for gallery presentation. Set href attribute for primary menu link "Gallery". Nesting dropdown menu you should remove.
    3. Choose suitable variant for news presentation. Set href attribute for primary menu link "News". Remove nesting dropdown menu.
    4. Remove unnecessary links in menu pattern.
    5. Remove from menu pattern link to page 404.
    6. Format order of menu elements relevantly to your needs.
    7. Copy received code of main menu on all pages of the template.
  4. Open files and replace content with the one you need.
  5. Customize plugins according to your needs. You can get more details in FAQs.
  6. Upload files on the server.

Some pages (like single page, news page, single movie page etc.) have 3 variant for posible sidebar layout: left sidebar, right sidebar, without sidebar (or full width page). You can choose most suitable for yourself.

If you need to add additional elements pay your attention to section "Shortcode and patterns". It demonstrates ready patterns which you can quickly adjust to your ideas.

gozha-nav.css

It contains styles for menu view. It has two formats of main menu view: a) full format (opened) menu – the menu that is used for large and medium displays (monitors, laptops, tablets (landscape orientation)); b) narrow format (compact) menu – styles used for small screens (phones, tablets (portrait orientation)). Transition point value is 768 pixels (is used as max-width condition for media query)


touch.css

Rules for touchscreen devices. The main purpose of this file is to provide certain hacks for mobile devices and to cancel animation used for desktop version.


style.css

Style.css is generated file of sets less files. It partly include bootstap 3.1.1 and all custom style rules.

Less files structure:

  • base group:
    • bootstrap-variables.less - collection of core bootstrap variables (read more );
    • vars.less - collection of custom variables (colors and fonts mostly);
    • normalize.less - a modern, HTML5-ready alternative to СSS resets;
    • bootstrap-type.less - bootstap core types for heading and typography;
    • bootstrap-scaffolding.less - a handful of variables for quickly customizing key elements of your site's skeleton;
    • bootstrap-mixins.less - bootstap function for speed up coding process; (read more )
    • base.less - custom rules for unclassed element. some most general rules. font init place;
  • layout group:
    • bootstrap-grid.less - bootstap 12-col grid system;
    • responsive-utilities.less - classes for toggling content visibility across viewport breakpoints;
  • object group:
    • alerts.less - default styles for bootstrap component - alert;
    • buttons.less - default styles for bootstrap component - button;
    • navbar.less - default styles for bootstrap component - tabs and accordions (partly);
    • navs.less - default styles for bootstrap component - tabs and accordions (partly);
    • panels.less - default styles for bootstrap component - tabs and accordions (partly);
    • component-animations.less - respond for smoothing effect on animated components;
    • objects.less - main file which contains styles for all patterts;
  • state group:
    • state.less - contains rules for all patterns variations and responsive block.

Lets take a look closely at our javascript files.


Part of the scripts is basic scripts and is used in all template pages. This includes:

jquery-1.10.1.min.js

jQuery is a multi-browser JavaScript library designed to simplify the client-side scripting of HTML.


jquery-migrate-1.2.1.min.js

This plugin can be used to detect and restore APIs or features that have been deprecated in jQuery and removed as of version 1.9. The plugin can be included with versions of jQuery as old as 1.6.4 to identify potential upgrade issues via its JQMIGRATE console warnings. However, the plugin is only required for version 1.9.0 or higher to restore deprecated and removed functionality.


jquery-ui.js

jQuery UI is a curated set of user interface interactions, effects, widgets, and themes built on top of the jQuery JavaScript Library. In this template it is used to make able functional kept in wijmo widgets.


modernizr.custom.js

Modernizr is a JavaScript library that detects HTML5 and CSS3 features in the user's browser.


bootstrap.js

Powerful mobile-first front-end framework for web development. It connects to the site through CDN "http://netdna.bootstrapcdn.com/bootstrap/3.1.0/css/bootstrap.min.css". You can get more information on official site http://getbootstrap.com/.The project uses range of elements (accordions, tabs, tooltips etc.) which are based on use of this framework. You can find description of these elements here


2 different sliders for realization main slider on home page. Revolution slider. Responsive jQuery Plugin. Find documentation here.


idangerous.swiper.js

Swiper slider. Swiper - is ultra lightweight mobile touch slider with hardware accelerated transitions (where supported) and amazing native behavior. Find documentation here.


Some specific plugins have been also used. Part of them use on each template page. Another part in some pages for specific needs.

jquery.mobile.menu.js

Designer’s plugin for mobile menu. It has two patterns of interaction with main menu: a) full format (opened) menu – the menu that is used for large and medium displays (monitors, laptops, tablets (landscape orientation)). b) narrow format (compact) menu – the menu which is used for displaying in small screens (phones, tablets(portrait orientation)).For mobile view initially menu looks like switch icon. After you open it every link is divided into two parts. Main part is a direct link to the menu item. Additional link (which is marked by plus icon in closed state and minus icon in open state) enables you to open context menu list. The menu is well adopted for touch devices since touchstart event is also performed.

jquery.form.js

The script presupposes 3 objectives. 1) Testing inputs placeholder availability and control of its corresponding display. 2) Checkup of input data in form inputs. If you face incompatibility (terms are not performed) shows error handler. 3) Data forming for sending message into php file. Using this data the message is crafted and then it will be sent.


form-element.js

Plugin helps generate custom form element like checkbox and radio button.


jquery.selectbox-0.2.min.js

Custom select box replacement inspired by jQuery UI source.


jquery.raty.js

jQuery Raty is a plugin that generates a customizable star rating. More information about setting you can find here


jquery.knob.js

Nice, downward compatible, touchable, jQuery dial. More information find here


count.down.js

A simple JavaScript API for producing an accurate, intuitive description of the timespan between two Date instances. Link


jquery.magnific-popup.min.js

Magnific Popup is a responsive lightbox & dialog script with focus on performance and providing best experience for user with any device. Plugin documentation have full information about customization.


addthis.js

AddThis - is a mobile-friendly sharing, following and content recommendations. This template uses share button for the most popular social networks: twitter, facebook, google plus, pinterest and linked in. If you need another particular set you can easily generate it. To generate share button you need to create an account on site www.addthis.com


maps.google.com/maps/api/js

InfoBox extends the Google Maps JavaScript API V3 OverlayView class.
An InfoBox behaves like a google.maps.InfoWindow, but it supports several additional properties for advanced styling. An InfoBox can also be used as a map label.
An InfoBox also fires the same events as a google.maps.InfoWindow.

How to work with main menu?

Main menu plugin initiates itself in file custom.js. It contains following options:

  • triggerMenu – tracks element if the code which performs switch role for mobile menu view.
  • subMenuTrigger – tracks elements serving for opening of dropdown menu in mobile view. (If your structure doesn’t provide dropdown menu this option is not necessary) .
  • AnimationSpeed – speed of hidden menu opening.

How to use megamenu?

Megamenu it’s a huge block which play role one of menu items. In this template it’s separate in 4 parts: each part for list represent. Each part contain nesting ul with list. They can be grouped and structured in different order. In general, it contains some important information and link to most useful pages. You can move this menu horizontaly. Just change value for "left" property in rules "ul#navigation > li > ul.mega-menu".
Megamenu fully responsive and you can use it for different dispay.

Twitter Feed

There are two steps to setup the Twitter Widget.

  1. Setup an application at Twitter dev center and get keys.
  2. Put your keys into Twitter files.
Setup Twitter application

  1. Visit https://dev.twitter.com/apps/ and sign in using your Twitter username and password. This doesn't have to be the username or password of the stream you need access to, just a Twitter account you control.
  2. Select 'Create new application' and enter the application details.
    1. The name and description can be anything you like really, but you can't use 'Twitter' in the name.
    2. The website field can be your main website and doesn't have to be the site where your Twitter feed or feeds are located.
    3. Callback URL can be left blank
  3. Enter the CAPTCHA info and click create
  4. On the next details screen, click 'create my access token'. You may need to refresh the page after a few seconds if it doesn't appear automatically.
  5. Make a note of the Consumer key, Consumer secret, Access token and Access token secret as highlighted below.
Setup Twitter application
  1. Open get-tweets.php file with help of your editor.
  2. Enter your twitter user name for $twitteruser = "";
  3. Enter your consumer key for $consumerkey = "";
  4. Enter your consumer secret key for $consumersecret = "";
  5. Enter your access token for $accesstoken = "";
  6. Enter your access token secret for $accesstokensecret = "";
  7. Open js/jquery.twitter.js file with help of your editor.
  8. Enter your twitter profile name for $twitterprofile = ""; (row 6)
  9. Enter your twitter screen name for $screenname = ""; (row 7)
  10. Save your changes.
